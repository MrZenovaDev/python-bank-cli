## 📖 About

A lightweight command-line banking app written in Python. No libraries, no setup — just run it and manage a bank account with full input validation and error handling.

## ✨ Features

- ✅ Check account balance anytime
- ✅ Deposit funds (rejects negative values)
- ✅ Withdraw funds (rejects overdrafts and negative values)
- ✅ Graceful error handling for invalid inputs
- ✅ Clean menu-driven loop — no crashes

## 📦 Installation

```bash
git clone https://github.com/MrZenovaDev/python-bank-cli.git
cd python-bank-cli
```

No dependencies — runs on plain Python 3.

## 🚀 Usage

```bash
python bank.py
```

```
1: Check balance
2: Deposit amount
```
